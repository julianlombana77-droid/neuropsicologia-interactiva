# Neuropsicología Interactiva (Módulos 5–8)

App educativa en **Streamlit** con:
- Conceptos clave por módulo (definición breve)
- Casos prácticos con retroalimentación inmediata
- Carga de nuevos casos vía JSON
- Descarga de definiciones/casos

## Ejecutar en local
```bash
pip install -r requirements.txt
streamlit run neuro_modulos_app.py
